# WELCOME TO JDNETTING

### HOW COULD YOU USE IT?
Steps:
- Enable execute permission
```
chmod +x running.sh
```
- Execute *running.sh*
```
./running.sh
```
- Write the red IP and the number of subnets you want.
```
jdnet 192.168.0.0 11
```

---
`Alias => jdnet` <br>
`Red IP => 192.168.0.0` <br>
`Quantity subnets => 11`
